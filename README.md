# ML4E-Club-Project
Project Title-
Virat Kohli The King Website

Description-
This project is a simple responsive website dedicated to Virat Kohli. The website showcases his career journey, major records and achievements, iconic images and an inspirational quote. The purpose of this project is to demonstrate basic frontend development skills using HTML and CSS with simple animations and responsive design.

Pages of the Website:
Home-
The home page introduces Virat Kohli with the title Virat Kohli The King. It contains an image, a short introduction and an About section that briefly describes his birth and early goals. It also includes an Explore Career button that leads to the career page.

Career-
The career page shows important milestones in Virat Kohli’s cricket journey. The information is displayed using cards that highlight key moments such as his international debut, World Cup victories, captaincy era and IPL success.

Records-
The records page presents some of the major achievements and statistics of Virat Kohli. The information is shown in card format similar to the career page to maintain consistent design.

Gallery-
The gallery page contains a collection of iconic images of Virat Kohli. It also includes a motivational quote.

Features:
Responsive design so the website looks good on mobile phones, tablets and laptops
Fixed navigation bar that remains visible while scrolling
Card hover animations for career and records sections
Button hover animation for the Explore Career button
Image animation on the home page
Simple and clean user interface
Technologies Used
HTML5
CSS3

Purpose of the Project-
This project was created as part of the Web Development Team Induction assignment for ML4E. It demonstrates basic skills in webpage structure, styling, responsive layouts and simple CSS animations.
